# Image Resizer Tool

This tool resizes all images in the `input_images` folder and saves the resized versions
into the `output_images` folder.

## How to Run

1. Install Pillow:

```
pip install pillow
```

2. Run the script:

```
python image_resizer.py
```

## Folder Structure

```
image_resizer_tool/
├── image_resizer.py
├── README.md
├── input_images/
│   ├── sample1.jpg
│   ├── sample2.jpg
│   └── sample3.jpg
└── output_images/
```

Resized images will be generated automatically.
